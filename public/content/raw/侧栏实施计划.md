**目标：** 将当前聊天区内的 Project Radar 胶囊入口改为右侧 `Todolist / Radar` 双 Tab，并让 Radar Tab 承载建议、反馈和项目弹窗入口。

**架构：** 新增 `WorkspaceSidePanel` 作为右侧栏外壳，复用现有 `TodoListPanel`，并用轻量 `RadarPanel` 展示当前单条建议。`RadarActionBar` 继续负责建议接受与反馈，但通过展示模式同时服务输入框上方和侧栏；数据查询、接受、反馈和双栏弹窗接口保持不变。

**技术栈：** React 18、Vite、Node.js 内置测试、现有 CSS Token、FastAPI。

## 1. 全局约束

- 右侧栏只新增一个 Tab，不新增独立页面或第二条侧栏。
- 输入框上方 Action Bar 只在 `pending` 状态展示；加载和接口错误不打扰主聊天。
- Radar Tab 可以展示加载、空状态和真实接口错误，并提供重试。
- 侧栏同一时刻最多展示一条最高价值建议；徽标值为 `0` 或 `1`。
- “查看全部项目”复用现有 `ProjectRadarModal`。
- 移除聊天区中的 `ProjectRadarButton`，不改后端 API。
